# Digital-Media-Marketing
Install xampp control from phpmyadmin website
Start the Apache and MySql from the xampp controll
Extract the .zip folder in this location C:\xampp\htdocs\Raptor-main
Enter this URL "http://localhost/phpmyadmin/" and import the .sql file in phpmyadmin server page
Now run the website using this URL "http://localhost/Raptor-main/index.html"
